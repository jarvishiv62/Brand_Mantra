# BRANDMANTRA — Digital Marketing Agency
## ब्रांड का मंत्र | The Mantra of Brands

A premium Indian digital marketing agency website built with **Laravel 11**, **Alpine.js**, and a rich Indian aesthetic featuring Sanskrit typography, gold/saffron/crimson design system, mandala motifs, and Devanagari script accents.

## 🚀 Local Development Setup

### Prerequisites
- PHP 8.2+
- Composer
- Node.js 18+
- PostgreSQL (or use SQLite for local)

### Steps

```bash
# 1. Clone & install PHP deps
git clone <repo-url> brandmantra
cd brandmantra
composer install

# 2. Environment setup
cp .env.example .env
php artisan key:generate

# 3. Configure database in .env
# For local SQLite (easiest):
DB_CONNECTION=sqlite
# DB_DATABASE=/absolute/path/to/database/database.sqlite

# For local PostgreSQL:
# DB_CONNECTION=pgsql
# DB_HOST=127.0.0.1
# DB_PORT=5432
# DB_DATABASE=brandmantra
# DB_USERNAME=your_user
# DB_PASSWORD=your_password

# 4. Run migrations
php artisan migrate

# 5. Install JS deps & build assets
npm install
npm run dev   # development with hot reload

# 6. Serve
php artisan serve
# Visit: http://localhost:8000
```

## ☁️ Render Deployment

### Option A: Auto-deploy with render.yaml (Recommended)
1. Push this repository to GitHub
2. Go to [render.com](https://render.com) → New → Blueprint
3. Connect your GitHub repo
4. Render will auto-detect `render.yaml` and set up:
   - Web Service (PHP)
   - PostgreSQL database
   - All environment variables

### Option B: Manual Render Setup
1. Create a **PostgreSQL** database on Render → note the connection details
2. Create a **Web Service**:
   - **Environment**: PHP
   - **Region**: Singapore (best for India)
   - **Build Command**:
     ```
     composer install --optimize-autoloader --no-dev && npm install && npm run build && php artisan key:generate --force && php artisan config:cache && php artisan route:cache && php artisan view:cache && php artisan migrate --force
     ```
   - **Start Command**:
     ```
     php artisan serve --host=0.0.0.0 --port=10000
     ```
3. Set environment variables:
   ```
   APP_NAME=BrandMantra
   APP_ENV=production
   APP_DEBUG=false
   APP_KEY=<auto-generated>
   APP_URL=https://your-app.onrender.com
   DB_CONNECTION=pgsql
   DB_HOST=<render-db-host>
   DB_PORT=5432
   DB_DATABASE=brandmantra
   DB_USERNAME=<render-db-user>
   DB_PASSWORD=<render-db-password>
   SESSION_DRIVER=file
   CACHE_DRIVER=file
   LOG_CHANNEL=stderr
   ```
